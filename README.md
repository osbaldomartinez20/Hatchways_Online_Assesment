# Backend-Challenge
```
Backend Challenge for a potential position at Hatchways.
```

### Requirements

Node 15.4.0
Nodemon

### Usage

```
1. In the terminal and within the project directory inside the server directory run 'npm install'
2. Run 'npm start' to start the server, which will be at localhost:3000 since we are using port 3000
3. Step 1 solution will be found at localhost:3000/api/ping
4. Step 2 solution will be found at localhost:3000/api/posts?tag=history,tech&sortBy=likes&direction=desc
Or to see diferent results just change the tag, sortBy, and direction values.
7. Run 'npm test' to see the passing tests in the terminal
```
